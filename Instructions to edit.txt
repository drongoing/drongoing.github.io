
● Two ways to do it — pick whichever fits your workflow:

  ---
  Option A: Edit directly on GitHub (no local setup needed)

  Go to github.com/drongoing/drongoing.github.io, navigate to any file, click the pencil icon to edit, make changes, and
   click Commit changes. The site rebuilds automatically within a minute.

  To publish a new blog post:
  1. Go to the _posts/ folder on GitHub
  2. Click Add file → Create new file
  3. Name it exactly: YYYY-MM-DD-your-post-title.md (e.g. 2026-02-18-my-new-post.md)
  4. Paste this at the top, then write below it:

  ---
  layout: post
  title: "Your Post Title"
  date: 2026-02-18
  excerpt: "One sentence summary shown on the blog listing page."
  ---

  Your content here, written in Markdown.

  ## A heading

  A paragraph. **Bold**, *italic*, [a link](https://example.com).

  5. Click Commit changes → live within ~60 seconds.

  ---
  Option B: Edit locally and push (better for larger edits)

  # Edit files in your local folder, then:
  cd "C:/Users/vibhu/Projects/Claude Projects/drongoing.github.io"

  Quick reference — what file controls what

  ┌────────────────────────────────────────────────┬────────────────────────────┐
  │            What you want to change             │            File            │
  ├────────────────────────────────────────────────┼────────────────────────────┤
  │ Bio, headshot, social links, projects, gallery │ index.html                 │
  ├────────────────────────────────────────────────┼────────────────────────────┤
  │ Site title, URL, metadata                      │ _config.yml                │
  ├────────────────────────────────────────────────┼────────────────────────────┤
  │ Nav, footer, page structure                    │ _layouts/default.html      │
  ├────────────────────────────────────────────────┼────────────────────────────┤
  │ Colors, fonts, spacing                         │ assets/css/main.css        │
  ├────────────────────────────────────────────────┼────────────────────────────┤
  │ A blog post                                    │ _posts/YYYY-MM-DD-title.md │
  ├────────────────────────────────────────────────┼────────────────────────────┤
  │ Blog listing page                              │ blog/index.html            │
  └────────────────────────────────────────────────┴────────────────────────────┘
